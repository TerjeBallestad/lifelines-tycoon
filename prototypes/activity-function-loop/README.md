# Activity Function Loop Prototype

Disposable HTML prototype for Lifelines Tycoon v2.

## What it tests

Can one obscure activity move through the playable loop?

```text
observed behavior
→ interpreted function
→ risky tiltak choice
→ watched effect
→ reviewed understanding
```

The prototype uses one activity: **Collects Gro clippings**.

## Resource model under test

- **Dice** are one-use state attention windows.
- **Activity menu** records observed behavior, known details, unknowns, working function, and practice paths.
- **Frank / Grete / observation / tiltak** are different ways of spending attention around the same activity.

## DSM Pokémon guardrail

The prototype avoids real clinical diagnosis language. It identifies functional roles like Security, Script, Control, Expression, and missing/brittle skills.

## How to open

```sh
open prototypes/activity-function-loop/index.html
```

## Deliberate limits

- No production Godot code.
- No persistence.
- No broad activity catalogue.
- No real random simulation.
- No final UI polish.

The point is to test whether understanding one activity makes the next desk decision more interesting.
