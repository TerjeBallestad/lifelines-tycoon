# Caseworker Talents — branching skill tree screen

## Anchor

A full-screen "Caseworker Talents" tab, accessible from the main menu. Branching node tree, three columns:

> **Empathy branch:** Active Listening I → Active Listening II → Trauma-Informed Communication (capstone)
> **Observation branch:** Sharp Eye I → Sharp Eye II → Pattern Recognition (capstone)
> **Bureaucracy branch:** Form Filing I → Form Filing II → Bureaucracy Mastery III (capstone)

Each node grants a passive: "Active Listening I: psych evaluations cost 0.5h less and yield +1 case-file entry." Players earn "Talent Points" by completing arcs; spending them is permanent and unlocks late-game Tiltak.

A tooltip: *"Specialize your caseworker! Choose a path that matches your playstyle."*

## Why this scores low on thematic-coherence

The skill tree is the second-most theme-destroying element after a player avatar. The play is no longer about Elling; it is about building "your caseworker." Late-game Tiltak unlocks attach to the *player*, not to Elling's case-file growth — the discovery loop inverts. Sub-criterion 1 floors because "your caseworker" + "playstyle" is hero-language. Sub-criterion 4 floors because the build IS an RPG progression frame, with talent points and capstones.

## Specific sub-criteria signal

- Sub-criterion 1 (Player role = state, not hero): 0 — "your caseworker" + "playstyle" + customizable specialization is the RPG hero pattern, dressed as a caseworker.
- Sub-criterion 4 (No RPG progression frame): 0 — skill tree, talent points, capstone abilities, permanent unlocks: every RPG-progression beat present.
